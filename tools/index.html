My AutoHotkey configuration script

Collection of my personal keyboard customisations for AutoHotkey, including

  • Tap Ctrl for Esc
  • Hold LCtrl for numpad input (jkl-uio to 123-456 etc.)
  • Mac-style dead key diacritic input (e.g. alt+i, a outputs â)
  • Useful glyphs for typography, e.g. quotation marks, ligatures, en vs em dashes
  • Use QWERTY layout for shortcuts while holding Ctrl or Win (instead of Dvorak)
